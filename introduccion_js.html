Eleméntos del lenguaje

  • Función -> Es una instrucción, se crea 'function nombre() {}' y puede contener parámetros o argumentos
  • Función Anónima-> Es una instrucción, se crea '() => {}' no contiene parámetros o argumentos
  • Metodo -> Se invocan despues de un punto, se escriben "camelCase" y contienen argumentos o parametros ej: document.getElementById(parametro)
  • String o Cadenas de texto -> Se representa siempre entre comillas "Soy un String".
  • Variable -> Espacio de memoria que almacena objetos y hacen referencia a un valor. JS es un lenguaje no tipado, ya que no hay que hacer referencia al tipo de valor que vá a contener nuestra variable
    // Varibles JS ej var numero = 100 var numero = 100.01 var numero = "Texto test" var numero = numero + 1 var otro = 13 var numero = numero + otro const siempre = 0 // ej en otros lenguajes de bajo nivel int numero = 100 float numero = 10.76 String numero = "Texto test" //
  • Arrays -> Son grupos de objetos, estos se encierran entre corchetes '[objeto1, objeto2, objeto3]'
  • Booleanos -> Una variable de tipo boolean almacena un tipo especial de valor que solamente puede tomar dos valores: 'true' o 'false'.
  • Operadores
    • Asignación: '='
    • Incremento y decremento: suma 1: '++', resta 1: '--'
    • Logicos: negación '!', And '&&', Or '||'
    • Matemáticos: Más: '+', Menos:'-', Entre:'/', Multiplicación'*', Modulo:'%' -> calcula el resto de la división entera de dos números. Si se divide por ejemplo 10 y 5, la división es exacta y da un resultado de 2. El resto de esa división es 0, por lo que módulo de 10 y 5 es igual a 0.
      Los operadores matemáticos también se pueden combinar con el operador de asignación: '+=', '-=', '*=', '/=,'%='.
    • Relacionales: Mayor que '>', Menor que '<', Mayor o Igual '>=', Menor o Igual '<=', Igual que '==', Distinto que '!='.
  • Estructuras de control (De flujo) -> los programas dejan de ser una sucesión lineal de instrucciones para convertirse en programas inteligentes que pueden tomar decisiones en función del valor de las variables.
    • 'if'
    • 'if else'
    • 'when'
    • 'for' -> for(inicializacion; condicion; actualizacion) {...}
    • 'for in' -> for(indice in array) {...}
    • 'switch'

La sintaxis de JS:

  • No se tienen en cuenta los espacios en blanco y las nuevas líneas: como sucede con XHTML, el intérprete de JavaScript ignora cualquier espacio en blanco sobrante.
  • Se distinguen las mayúsculas y minúsculas.
  • No se define el tipo de las variables, no es necesario indicar el tipo de dato que almacenará.
  • No es necesario cerrar linea con punto y coma (;).
  • Existen dos tipos de comentarios: los de una sola línea y los que ocupan varias líneas. Ejemplo: // ó /**/
